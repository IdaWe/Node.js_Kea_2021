Visual Studio code er et lightweigt development tool

Under overfladen:
Visual Studio Code er bygget på noget der hedder electron som igen er bygget på Node.js
Crome er bygget på chromeom
Chromeoum er en open source browser
Chrome bruger en javascript engine der hedder v8 engine (som er meget hurtig)

En Dansk gut opfandt et shared spaced for varibles så der skulle mindre machine instructions til at accesse variablerne og derfor hurtigere til at execute. Og det er det der I chrome
Før node var det eneste sted man kunne køre javascript I browseren
Node var en ide hvor man sagde hvad hvis vi tog v8 engine ud og repackaged den så den blev dens eget solo inviroment.
Og det er det Nodejs er. Det er et invironment der tillader at man køre javascript udenfor browseren - det er den simpleste definition. Nodejs lader dig køre serverside javascript (og det er en helt ny ting)
Nu når vi ved vi kan køre javascript alle steder og ikke kun for clienten I browseren er en af tingene vi kan create applications, fx desktop apps
Electron er et framework som VS code køre på
Electron bruger node som er med til at tillade at man kan lave applikationer; twitch, messenger osv. bruger electron
Man kan også create fullstack websites
