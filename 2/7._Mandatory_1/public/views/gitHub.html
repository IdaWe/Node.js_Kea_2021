For at clone et repository skriver vi git clone + URL og så ligger det der hvor vi står, fx desktop
Vi cd'er altså til hvor vi vil have det og henter det så
Når vi så efterfølgende vil have de opdateringer han har lavet går vi ind I mappen og skriver så git pull, så henter den fra repository'et online og ned I mappen


Når vi skal adde til git:
git add -A for at adde alt git commit -m Så en commit message git push Så pushes det til git Pushe noget til git via terminal: - Gå ind I dit repo først - git add -A (den adder alt, det der ik skal addes er I min git ignore) - git comit -m "bla bla" (adder comit message) Git push Pulle noget fra git via terminal: - Først cloner du repository til fx desktop - Du står I dit projekt. Gå cd så til directory du vil pulle fra Skriv så git pull Git clone + url Git pull + url?
