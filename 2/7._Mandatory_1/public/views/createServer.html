Create a new server / API

Sådan sætter vi en server op
- Opret ny project folder
- Lav så the project file: package.json
○ Når vi har med json at gøre er det altid key value pairs.
§ Key er dependencies
§ Value er et object
○ Der skal altså stå I package.json:

○ Vi kan selv slå version op men vs code autocompleter for os (han har vist os siden hvor man kan se version: søg på google npm express)
- Nu efter vi har manuelt defineret express som en dependency I package.json vil vi gerne installere express.
- Men ikke bare express. Vi vil gerne installere alle dependencies defineret I package.json. For når man henter en fremmeds repo vil man gerne installere alt. så når vi har en færdig package.json siger man bare npm install:
○ npm install
- Lav nu en app.js file
○ At kalde den app giver mening fordi det er en application file/server
○ Og fordi documentation kalder den det (nok bedste grund)
○ Vi kan sætte den igang med køre filen allerede nu med nodemon.cmd app.js
○ Nu vil vi så gerne importere det library vi har inkluderet (express)
